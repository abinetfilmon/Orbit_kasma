# 🛰️ Orbit × Kasma — 7-Week Post Tracker

Social media post tracker dashboard for **Orbit Innovation Hub** and **Kasma Digitals**.

## Project Overview
- **25 enterprises** tracked across 3 platforms
- **TikTok · Facebook · Telegram** — 1 post per platform per week
- **7 weeks** = 525 posts per enterprise = **13,125 total posts**

## Features
- ✅ Manual post status tracking (tap to toggle)
- 📋 Weekly tracker with Week 1–7 tabs
- 🗓 Full 25×7 grid overview
- 📊 Analytics: platform stats, weekly bar chart, leaderboard
- ⚙️ Configurable start date, project name & consultant name
- 💾 Auto-saves to browser localStorage
- ⬇️ CSV export
- 🌙 Dark / Light mode

## Tech Stack
- React 18 + Vite
- No external UI libraries — pure CSS-in-JS
- localStorage for persistence

## Run Locally
```bash
npm install
npm run dev
```

## Deploy to Vercel
Push to GitHub, import repo in [vercel.com](https://vercel.com), deploy. Done.
